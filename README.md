Даныый сервер реализован на Node.js с использование Express, в качестве ORM используется Sequelize и база данных PostgreSQL.

Модель пользователя содержит следующие данные:
    - ФИО.
    - Дата рождения.
    - Email - уникальное значение.
    - Пароль.
    - Роль - admin либо userю.
    - Статус пользователя - активный или нет.
    - Стату блокировки пользователя. 

Реализованы следующие endpoint:
1) Регистрация пользователя.
2) Авторизация пользователя с использованием JWT токена.
3) Пользователь может получить свои данные по ID (Данные любого пользователя может получить админ), получить данные можно после авторизации.
4) У администратора есть список всех пользователей, реализован постраничный вывод.
5) Блокировка пользователя. Пользователя может заблокировать сам себя (после блокировки будет перенаправлен на главную страницу). Администратор может заблокировать любого пользователя.

К данному серверу реализован Frontend, на React, в данном приложении можно проверить работу реализованых endpoint, приложение доступно по адресу 
https://test-tasks.ru 
    Тестовые пользователи:
    Админ - test@mail.ru, пароль 123.
    Пользователь заблокированный - test1@mail.ru, пароль 123.
    Пользователь с открытым доступом - test2@mail.ru, пароль 123.
В приложении есть две страницы - пользователя и администратора.

Серверная и фронтенд части загружены на выделенный сервер, в качестве прокси сервера используется nginx, установлен ssl сертификат. Также, на сервер установлена база данных PostgreSQL.
